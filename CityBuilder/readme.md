# Starter Project 
## p5.js with Typescript
Project to quickly get something working in [p5.js](https://p5js.org/) and [typescript](https://www.typescriptlang.org/)

## Getting Started

### Installing

```
git clone https://github.com/Gaweph/p5-typescript-starter.git
npm install
```

### Usage

```
npm start
http://localhost:3000
```
